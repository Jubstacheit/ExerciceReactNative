To run app :

- go to https://expo.dev/accounts/jubstacheit/projects/ExerciceReactNative/builds/0f37307a-110c-4285-bdb1-c195c557cb32 on Android device and install "Build artifact"

or

- run "npm install" in directory, then "npm run start --web" for web version

or

- run "npm install" in directory, then "npm run start --android" to run in emulator, or scan QR code in Expo Go
